Abstract

Arabic calligraphy represents one of the richest visual traditions of the Arabic language, yet current benchmarks for multimodal large language models (MLLMs) largely overlook this domain. We introduce DuwatBench, a comprehensive benchmark of 1,050+ curated samples spanning six classical and modern calligraphic styles: Thuluth, Diwani, Kufic, Naskh, Ruq'ah, and Nasta'liq.

Unlike existing Arabic OCR datasets that focus on clean printed or handwritten text, DuwatBench targets real-world calligraphy images featuring complex artistic backgrounds, dense ligatures, and stylistic variations that challenge standard recognition systems.

Each sample includes full text transcription, style labels, theme categories, and bounding box annotations for individual text instances. We evaluate 13 state-of-the-art models—8 open-source and 5 closed-source—using five complementary metrics with Arabic-aware normalization.

Our results reveal significant challenges: the best-performing model (Gemini-2.5-flash) achieves only 56.1% exact match accuracy, with most models showing CER above 60%. These findings highlight a critical gap in current MLLMs' ability to handle culturally significant visual text.

Dataset Overview

Data Statistics

  • Total Samples: 1,050+ images
  • Unique Words: 1,400+
  • Calligraphy Styles: 6
  • Theme Categories: Religious & Non-religious
  • Annotations: Text + Bounding Boxes

Evaluation Modes

  • Full Image: Recognize all text in image
  • With BBox: Recognize text within bounding boxes

Metrics

  • Character Error Rate (CER)
  • Word Error Rate (WER)
  • chrF Score
  • Exact Match Accuracy
  • Normalized Levenshtein Distance (NLD)
